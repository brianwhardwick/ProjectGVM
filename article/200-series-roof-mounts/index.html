200 Series RCP Roof Mounts Additional Roof Mounts Using RCP Bases

The Problem

The Project GVM 200 Series required a rooftop tent. The common solution for the 200 Series is a backbone-style continuous rail system supporting either a roof platform or load bars.

However, this build was not driven by convention alone. One of the primary objectives was to retain the ability to quickly remove the rooftop tent, allowing the vehicle to remain lower, lighter, and less visually intrusive for long-distance touring without camping gear or for more technical 4WD trips where reduced roof mass and height are beneficial.

Rather than selecting a roof system based purely on what is most commonly fitted, the setup was approached as an engineering problem with defined constraints and trade-offs.

Design Criteria

The following criteria were established before selecting a final roof mounting arrangement:

  • Maintain manufacturer roof load limits
  • Minimise constant roof mass
  • Minimise centre of gravity increase
  • Enable quick and easy rooftop tent removal
  • Maintain a clean, low-profile roof when the tent is removed
  • Distribute load across available mounting points where possible
  • Retain compatibility with load bars and platforms

This reflects the same methodology used in the Project GVM calculator, where modifications are assessed as part of the complete vehicle system rather than in isolation. Small changes in component mass, position, and mounting height can compound quickly and materially affect axle loads, GVM compliance, and overall vehicle dynamics.

Options Considered

The main roof mounting options considered were:

  • Backbone with platform: Excellent roof rating, but the platform adds significant constant mass before any payload is applied. This reduces available GVM and increases roof load demand.
  • Backbone with load bars: Strong and proven, but remains visually intrusive and permanently prominent even when the rooftop tent is removed.
  • Rhino-Rack RCP roof mount kit: Lower mass, adequate roof rating (150kg), and a minimal footprint when removed, which aligned more closely with the design criteria.

The standard RCP roof mount kit utilises six of the eight available roof mounting points. For this application, that created two concerns: first, load was being introduced through fewer attachment points than the roof physically allows, and second, the mounting layout resulted in an undesirable front overhang with the rooftop tent installed.

While the standard RCP system met many of the design objectives, it was clear there was scope to refine the configuration further. This is consistent with the broader Project GVM approach: not simply fitting parts, but understanding how a modification behaves as part of the whole vehicle system and improving it where practical.

The Solution

Further research found limited documented examples of alternative RCP mounting arrangements. Eventually, an older reference was identified showing a Rhino-Rack kit configuration using two sets of the rear mounts, and a similar arrangement also appeared in a now-discontinued package from a roof rack supplier.

Using the three rear mounts installed and checking alignment with a straight edge, it was confirmed that a rear mount can be used in the forward position while maintaining the correct mounting angle. The only notable difference was that the forward-installed rear mount sat approximately 5mm lower than the other bases.

Rhino-Rack supply off-the-shelf spacers up to 20mm, so correcting the 5mm height difference was straightforward. All additional components required to achieve this setup are shown below.

This does not formally increase Rhino-Rack’s published roof rating of 150kg, and it should not be interpreted as doing so. However, increasing from six to eight mounting points improves load sharing across the roof. Assuming broadly even distribution, load per mounting point is reduced by approximately 25%, which lowers localised stress in the bases, fasteners, and roof structure while also improving structural redundancy.

From an engineering perspective, the benefit is not a new rating, but better distribution of the same load across more available attachment points. For a system carrying constant elevated mass such as a rooftop tent, reducing concentrated load paths is a worthwhile improvement.

Rhino-Rack’s standard leg option for the 200 Series is the RCH leg at 52mm height. As the Project GVM 200 Series does not have a roof-mounted aerial, RCL legs at 37mm height were used instead, reducing the installed height by 15mm.

Again, this does not formally change the roof rating. However, a lower mounting height reduces the load’s moment arm relative to the roof structure. In practical terms, this reduces bending moments under dynamic loading, lowers aerodynamic drag and wind exposure, and minimises additional centre of gravity increase. For any roof-mounted mass, particularly a rooftop tent, these are worthwhile gains.

This configuration also results in a lower profile than a backbone system using the same load bars, while remaining compatible with Pioneer platforms if required.

System-Level Considerations

Roof-mounted accessories should not be considered in isolation. A rooftop tent, load bars, and mounting system all contribute constant elevated mass, which affects more than just the roof. The additional weight also influences axle loading, body roll tendency, and overall vehicle compliance.

This is the same reason the Project GVM calculator assesses modifications as a full vehicle system. A change that appears minor in isolation can have a measurable effect once combined with fuel, passengers, cargo, drawer systems, bull bars, or towing loads. Roof weight is especially important because it is both elevated and permanent while fitted.

In other words, this setup was not chosen simply because it fit. It was selected because it better satisfied the overall design criteria: minimise unnecessary mass, improve load distribution, reduce overhang, and retain modularity.

Validation

The final configuration was validated through a straightforward but deliberate process:

  • Mount alignment checked using a straight edge across installed bases
  • Spacer height selected to correct the measured 5mm discrepancy
  • Load bar fitment checked to confirm consistent engagement across all mounting points
  • Final arrangement assessed with rooftop tent installed
  • Real-world use observed under touring and off-road conditions

No abnormal movement, obvious misalignment, or excessive deflection has been observed in service. While this is not a substitute for manufacturer certification, it provides confidence that the modified arrangement functions as intended within the original published roof load limits.

Final Result

The final result is shown below: eight bases installed, heavy duty load bars fitted, and rooftop tent mounted in a configuration that is lower, cleaner, and better distributed than the standard six-mount arrangement.

The front load bar position still allows sufficient clearance for the sunroof to open.

If a rooftop tent is not fitted, a platform mounted on the RCL legs provides a very low-profile option, lower than a backbone system with comparable bar and platform height. This arrangement was used on the Project GVM vehicle before the rooftop tent was installed.

Conclusion

The final eight-mount RCP configuration does not change the published roof rating, but it better satisfies the engineering objectives of this build than the standard six-mount arrangement. It reduces front overhang, improves load distribution, lowers installed height, and retains the modular low-profile roof setup that was a key project requirement from the start.

As with all modifications, the roof system should be considered as part of the complete vehicle loading system rather than as an isolated accessory. That same thinking underpins the Project GVM calculator: understand the effect of each change, quantify where possible, and build solutions thoroughly rather than by halves.
